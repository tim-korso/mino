# Safety Dimensions — 五维通用安全框架

> 从「护肤品成分清单」升级为「五维通用框架」。
> 不再硬编码品类的安全信号——用维度当 checklist，Phase 0.3 搜索品类具体风险。

## 为什么需要通用框架

原有安全红线只覆盖护肤品/化妆品场景：

| 原信号 | 覆盖品类 |
|--------|---------|
| 含强效激素 | 护肤品 |
| 孕妇/哺乳禁用成分 | 护肤品/补剂 |
| 已知不可逆副作用 | 护肤品/药品 |
| 成分不透明+功效强宣称 | 护肤品/补剂 |
| 禁用药/违禁成分 | 药品/补剂 |

纺织品的安全问题（甲醛、AZO 染料）一个都捕获不到。电子产品（电池安全、隐私）也没有。

五维框架让 Phase 0.3 按维度逐项检查，不依赖预存知识。

## 五维总览

```
化学安全 — 产品含不含有害化学物质？
物理安全 — 产品会不会造成物理伤害？
生物安全 — 产品会不会滋生/传播病原体？
数据安全 — 产品会不会泄露隐私或数据？
财务安全 — 购买/使用有没有隐性成本或欺诈？
```

## 各维度详细定义 + 品类信号速查

### 1. 化学安全

**定义**：产品是否含有已知对人体有害的化学物质——致癌物、内分泌干扰物、致敏物、重金属等。

**常见信号**：

| 品类 | 已知风险 | 验证方式 |
|------|---------|---------|
| 纺织品/服装 | 甲醛（防皱定型剂）、AZO染料（致癌芳香胺）、重金属（染料/辅料）、烷基酚聚氧乙烯醚（APEO） | OEKO-TEX Standard 100, GB 18401 |
| 护肤品/化妆品 | 禁用防腐剂（甲基异噻唑啉酮等）、激素（丙酸氯倍他索等）、重金属（汞/铅美白产品） | NMPA 备案查询、成分表核对 |
| 食品 | 农药残留、兽药残留、重金属、非法添加（苏丹红/三聚氰胺/…） | SC 编号查询、第三方检测报告 |
| 补剂 | 处方药非法添加（西布曲明/西地那非等）、重金属超标 | GMP 认证、第三方检测（ConsumerLab/USP） |
| 电子产品 | 阻燃剂（PBDE）、RoHS 违规（铅/汞/镉）、邻苯二甲酸盐（线材） | RoHS 认证、FCC/CE 认证 |
| 家具/装修 | 甲醛释放（板材/胶水）、VOC（油漆）、氡（石材） | GB 18580（板材甲醛）、CARB 认证 |
| 玩具/儿童用品 | 邻苯二甲酸盐（塑化剂）、重金属（涂料）、小零件 | GB 6675, EN 71, ASTM F963 |
| 不粘锅/厨具 | PFAS/PFOA（特氟龙涂层）、重金属迁移（陶瓷涂层） | FDA/EFSA 食品接触材料标准 |

**搜索模板**：`"[品类/材质] 有毒 有害物质 超标 检测"`

### 2. 物理安全

**定义**：产品是否存在结构缺陷、电气隐患、火灾风险、机械伤害等物理层面的安全威胁。

**常见信号**：

| 品类 | 已知风险 | 验证方式 |
|------|---------|---------|
| 电子产品 | 电池鼓包/起火/爆炸、过热烫伤、接地不良/漏电、尖锐边缘 | UL 认证、CCC 认证、召回记录查询 |
| 汽车 | 碰撞安全、制动失效、气囊缺陷、座椅安全带缺陷 | NCAP 评级、IIHS 评级、NHTSA/市监局召回公告 |
| 玩具/儿童用品 | 小零件窒息风险、锐利边缘/尖端、折叠机构夹伤、弹射物伤害 | GB 6675 玩具安全、ASTM F963 |
| 家具 | 倾倒风险（柜子/抽屉柜未固定）、结构不稳固 | ASTM F2057（家具倾倒）、GB 28007（儿童家具） |
| 工具/器械 | 防护罩缺失、飞溅物伤害、振动伤害 | CE 机械指令、ISO 13849 安全相关控制 |
| 运动器材 | 结构断裂、防护不足 | EN 957（健身器材）、ASTM F2276（健身车） |

**搜索模板**：`"[品类] 召回 安全事故 起火 爆炸 断裂 受伤"`

### 3. 生物安全

**定义**：产品是否存在微生物污染、过敏原、致病菌等生物性风险。

**常见信号**：

| 品类 | 已知风险 | 验证方式 |
|------|---------|---------|
| 食品 | 菌落总数超标、致病菌（沙门氏菌/金黄色葡萄球菌/…）、霉菌毒素 | SC 编号、批次检测报告、HACCP 认证 |
| 补剂 | 微生物污染、原料变质 | GMP 认证 |
| 化妆品 | 菌落超标（开封后）、致敏成分 | NMPA 备案、斑贴测试 |
| 纺织品 | 致敏染料、抗菌剂引起的皮肤菌群失衡、潮湿环境霉菌滋生 | OEKO-TEX Standard 100 |
| 宠物食品 | 沙门氏菌、黄曲霉毒素 | AAFCO 标准、FDA 召回记录 |
| 水处理/净水器 | 菌落二次污染（滤芯过期） | NSF/ANSI 认证 |

**搜索模板**：`"[品类] 菌落超标 过敏 污染 召回"`

### 4. 数据安全

**定义**：产品是否存在隐私泄露、数据滥用、安全漏洞等信息安全风险。

**常见信号**：

| 品类 | 已知风险 | 验证方式 |
|------|---------|---------|
| 智能家居/IoT | 默认密码/无密码、明文传输、云端数据泄露、未修补漏洞 | 固件版本检查、CVE 漏洞库查询、隐私政策审查 |
| 手机 App | 过度索取权限、后台数据上传、SDK 数据收集 | 权限列表审查、隐私政策、MobSF 等静态分析 |
| 智能摄像头/门锁 | 视频流被第三方访问、弱认证机制 | 加密协议审查、是否支持本地存储 |
| 汽车（联网） | 远程控制漏洞、位置数据泄露、驾驶数据被第三方使用 | 隐私政策、CVE 查询 |
| 健康设备/可穿戴 | 健康数据被出售/泄露、未加密传输 | HIPAA 合规（美国）、隐私政策、GDPR |
| 语音助手 | 持续录音、录音数据被人工审核 | 隐私政策、是否支持本地处理 |

**搜索模板**：`"[品类/品牌] 隐私 数据泄露 安全漏洞 CVE"`

### 5. 财务安全

**定义**：购买和使用过程中是否存在隐性成本、不公平条款、欺诈风险。

**常见信号**：

| 品类 | 已知风险 | 验证方式 |
|------|---------|---------|
| 房产 | 产权不清晰、隐性费用（维修基金/税费/中介费）、开发商资金链断裂/烂尾、虚假宣传（学区/面积） | 预售许可证查询、不动产登记查询、开发商信用查询 |
| 金融产品 | 条款陷阱（高额赎回费/提前还款罚金）、收益率误导（年化 vs 预期 vs 历史） | 合同条款审查、监管机构查询（银保监会/证监会） |
| 订阅服务 | 自动续费难以取消、免费试用后的隐性扣费 | 取消流程测试 |
| 保险 | 免赔条款不透明、理赔困难历史 | 监管投诉记录查询 |
| 二手车 | 事故车/泡水车/调表车、抵押车无法过户 | 维保记录查询、第三方检测（查博士/TÜV） |
| 预售/众筹 | 延迟交货、项目搁浅、虚假众筹 | 平台保障政策、发起方信用记录 |

**搜索模板**：`"[品类/品牌] 投诉 维权 陷阱 隐性消费 骗局"`

## Phase 0.3 执行模板

对当前品类，按五个维度各做一次搜索检查：

```
□ 化学安全：[品类] 有毒 有害物质 超标 → 有没有信号？
□ 物理安全：[品类] 召回 安全事故 → 有没有信号？
□ 生物安全：[品类] 菌落超标 过敏 污染 → 有没有信号？
□ 数据安全：[品类] 隐私 数据泄露 → 有没有信号？（仅电子/智能产品）
□ 财务安全：[品类] 投诉 维权 陷阱 → 有没有信号？
```

没有信号 ≠ 没有风险——可能是品类太新还没暴露问题。对新兴品类，安全标准可能尚未建立——这是最高级别的风险信号。

## 安全红线硬拦截（保留原有规则）

命中以下信号 → 直接 🔴，不进入后续管线：

| 信号 | 适用品类 | 例子 |
|------|---------|------|
| 含禁用/违禁化学物质 | 所有 | 纺织品→可分解致癌芳香胺染料超标 |
| 已知不可逆安全风险 | 所有 | 电池设计缺陷→起火无法扑灭 |
| 成分/材质不透明 + 强功效/安全宣称 | 所有 | "秘方材料"+"绝对无害" |
| 被主要市场监管机构禁售/召回 | 所有 | FDA/市监局/NMPA 禁售令 |
| 系统性财务欺诈证据 | 所有 | 已被多地消费者协会/监管机构警告 |

> 安全红线和品类内容类型判定并行——红灯亮了，品质再好也不往下走。
