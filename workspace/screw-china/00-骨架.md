# 螺丝在中国 · 骨架 v1

> 不是教你赢。是帮你看清——你在哪、谁在拧你、为什么拧、以及松动的可能性。

## 核心论点

**每个螺丝都在给定约束下做出最优选择——但这些约束不是自然存在的。它们由组织考核、制度设计、国家治理逻辑和全球体系逐层设置。"看清自己是螺丝"这个动作本身，就是在松动第一颗螺丝。**

## 跨层传导机制

| 传导方向 | 具体机制 |
|---------|---------|
| **全球 → 国家** | 贸易规则（WTO/关税）→ 产业政策空间；金融体系（美元/SWIFT）→ 货币政策自由度；技术管制（实体清单）→ 发展路径 |
| **国家 → 制度** | 法律框架 → 制度创设边界；财政分配（分税制/转移支付）→ 地方创新天花板；运动式治理 → 制度可预期性被周期性打破 |
| **制度 → 组织** | 考核指标（GDP/维稳）→ 组织指挥棒；编制控制 → 规模边界；晋升锦标赛 → 同级零和竞争 |
| **组织 → 个体** | KPI → 行为被量化捕获；晋升阶梯 → 时间偏好扭曲；身份区隔（编制内外/城乡）→ 风险偏好平行世界 |
| **元层次 → 个体** | 历史意识揭示路径依赖的非必然性 + 批判意识揭示规则的人造性 → 打开"别的拧法"的可能性空间 |

## 本书结构

### Ch1 螺丝的牢骚：被安排的人生
§1.1 汤姆的信用社——一个解剖样本
§1.2 "选了最优的路"——理性选择为什么通向不自由
§1.3 差序格局中的自我——费孝通的透镜
§1.4 螺丝的三种自我叙事：内卷·躺平·佛系
§1.5 牢骚不是消极——它是结构意识的起点

### Ch2 考核即模具：组织如何制造标准件
§2.1 KPI 的哲学——可量化的被重视，不可量化的被蒸发
§2.2 编制的物理学——螺丝的螺纹是怎么车出来的
§2.3 晋升阶梯的数学——金字塔形的漏斗
§2.4 周雪光的透镜——控制权分配理论
§2.5 当算法成为新的螺丝刀——平台劳动的组织控制

### Ch3 规则的明暗两面：潜规则不是制度的例外
§3.1 吴思的发现——合法伤害权是"找人办事"的经济学基础
§3.2 潜规则的生产机制——为什么"照章办事"永远不够
§3.3 明暗之间的穿梭——基层螺丝的日常生存智慧
§3.4 数字化能消灭潜规则吗——技术赋权的新形态
§3.5 信用社实录——汤姆看到的规则双轨

### Ch4 钟摆的两端：一统体制与有效治理的永恒张力
§4.1 周的诊断——超大规模国家的根本矛盾
§4.2 黄仁宇的追问——为什么中国没有"数目字管理"
§4.3 钟摆的历史轨迹——从计划到市场到新时代
§4.4 财源即权力——分税制怎样重新定义了中央-地方关系
§4.5 螺丝眼中的钟摆——每次摆过来发生了什么

### Ch5 全球齿轮压下来：国际体系如何限定国内选择空间
§5.1 世界体系中的中国——核心、半边缘还是边缘？
§5.2 美元的枷锁——SWIFT·美债·人民币国际化的困局
§5.3 技术的铁幕——芯片·实体清单·自主可控的代价
§5.4 贸易的紧箍——从入世接轨到脱钩重构
§5.5 中国拧回去的空间——非对称相互依赖中的杠杆

### Ch6 历史的重量：为什么今天的约束看起来像"自古以来"
§6.1 孔飞力的发现——制度在危机中复制自身而非变革
§6.2 黄仁宇的纵深——"无关紧要的一年"如何决定三百年的轨迹
§6.3 分税制 1994——一个历史选择的解剖
§6.4 路径依赖的铁律——越往后改变的成本越高
§6.5 螺丝为什么感觉不到被拧上的那个瞬间

### Ch7 看清螺丝：认知本身是第一颗可以被拧松的螺丝
§7.1 福柯的启示——权力不是被"持有"的，是分布式的
§7.2 看到结构=已经不在结构中——认知的距离效应
§7.3 被拧→理解拧→反向拧——螺丝的行动光谱
§7.4 当足够多的螺丝同时看清——集体意识的社会动力学
§7.5 最后的问题——不是"怎么逃出去"，是"你想拧成什么样"
