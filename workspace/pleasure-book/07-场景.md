## 7.1 三速——三种完全不同的做爱

（详见原 Ch6 §6.1 的内容——保留并增强）

### 15 分钟快节奏——会做和不会做的区别

快 ≠ 粗糙。秘诀：砍掉不必要的转换，不是砍掉连接。

- 润滑液从一开始就在手边——不是"等下找"
- "快"不意味着跳过前戏——意味着前戏 = 接吻 + 抚摸 + 润滑液 + 直接进入刚才在 Ch3-4 学会的最有效的刺激方式
- 一个人先到——手/口帮对方也到——不是"我先到=结束"

### 3 小时慢节奏——奢侈是"停留"

- 0-30 min: 完全不做生殖器刺激——接吻、抚摸、按摩——目标：让 SIS 刹车完全松开
- 30-60 min: 全身探索——停下来喝水、聊天、再继续——你有一个晚上
- 60-120 min: 进入 + 体位变化 + 高潮 2-3 次——如果身体想的话
- 120-180 min: 事后不结束——吃完东西、洗澡、接着聊

### 中等节奏（30-60 min）——长年累月的默认模式

维持新意的秘诀不是"每次换新姿势"——是**微变化**：在沙发上开始、这次从口交开始、这次不进入而用玩具、这次先说一个幻想。[T021]

## 7.3 当事情不按计划时

（详见原 Ch6 §6.4 的内容——保留核心）

### 软了、进不去、到不了——怎么不掉下床

**软了**：正常。酒精后、压力大、年龄>40、换体位时都可能。不要叹气/道歉过度/"你不行了"——这些把一次生理事件变成心理创伤。"没关系——我们做别的。"

**进不去**：加润滑液——不是"你的问题"。盆底痉挛→需要盆底物理治疗不是"忍忍"。

**到不了**：放弃"必须高潮"的目标→享受此刻。焦虑→SIS↑→越努力越到不了。

## 7.4 沟通——内置于场景，不是独立技能

> 第一版骨架把"沟通"作为独立章（Ch5）。经过经典映射——沟通不是独立骨——它是所有场景中内嵌的层。以下不是"沟通的技巧"——是"在场景中怎么说话"。

### 在性中说话：操作指令 vs Dirty Talk

```
操作指令（任何场景都可以用的）:
  "轻一点"  "重一点"  "就在那"  "别停"  "慢一点"  "现在快一点"
  → 这些不是 dirty talk——是实时导航
  → 不多余——对方没有你的身体的实时读数

Dirty Talk (如果你和对方都喜欢):
  → 没有标准脚本——说你喜欢和对方喜欢听的
  → 如果不知道——先问。不是在性进行中——是在中立的时刻:"你喜欢被叫什么/被说什么"
```

### 在中立时间（不在床上）谈困难的话题

（详见原 Ch5 §5.2-5.3 的内容——保留核心）

**"我想要 X"怎么说**：具体+积极+引导。不是在床上说——是在中立的时刻。"上次你那样画圈的时候——特别舒服"。

**收到反馈怎么不防御**：ta 在给你一张地图——不是批评你开车不好。

**被拒绝怎么不惩罚对方**：微表情管理。"没关系"——这三个字是真的。如果总是被拒绝——留到中立时间谈——不是在被拒绝的那一刻。

### 同意——不是"破坏气氛"——是"创造安全"

```
"我想..." → 对方知道接下来会发生什么 → 放松 → 不用防着你
"你想..." → 你关心对方的状态 → 安全感 → SIS刹车↓
"如果你不想——告诉我" → 有退出的自由 → 更愿意探索
```

肯定的同意（"只有 Yes 才算 Yes"）不会破坏气氛——如果你做得自然。它是"我尊重你的边界——所以你可以放心探索"。[T022]

## 7.5 当欲望不匹配时——协商不是妥协

（详见原 Ch5 §5.5 和原 Ch6 的关系内容——保留核心）

欲望差异不是一个人"有病"——是两个人的系统交点。协商不是"各退一步"——是创造第三种方案。频率不是唯一可协商的——"什么样的性""谁发起""做完之后什么感觉"——这些全部可以重谈。

### 长期关系中的性——不只是"做多少次"

维持长期关系中性满意的因素（详见性健康书 Ch6 的研究支持）：
1. 继续把性作为优先项——不是"所有事做完之后"
2. 扩展性的定义——不只有阴茎-阴道性交
3. 各自维护身体和心理健康
4. 日常的非性身体接触——"没有性目的的触摸" = 低水平 SIS——是防止欲望消失的最强日常保护 [T023]

## 7.6 创造一个"性-friendly"的生活

性的质量不是某一次决定的——是生活方式的累积（详见性健康书五根骨头的生理基础）：

```
运动 → 耐力+血管健康+体象自信
睡眠 → <6h = 睾酮↓10-15%。>7h = 基线恢复
压力管理 → SIS 刹车的日常水平
非性亲密 → 日常没有性目的的触摸 = 低水平 SIS
性沟通 → 不只是"在性时说"——是中立时刻的正常话题
```

**一次很好的性 = 当天没有被踩刹车的因素 + 充足的非性连接 + 双方都知道自己的身体要什么 + 有一个让对方保持"一个独立的人"的方式**
