## 附录B：争议图鉴——我们不知道什么 ★诚实标记

以下是性医学领域最大的未决争议。每一条都有两边的高质量证据。

### 争议一：G点——解剖结构还是主观体验？

```
"G点存在"侧:
  → 阴道前壁1/3处有敏感区——很多女性报告刺激此处带来快感
  → 解剖学上：阴蒂-尿道-阴道复合体在此处汇集（阴蒂内部结构+尿道旁腺+阴道壁）
  → 有研究用超声显示：声称有"阴道高潮"的女性此处组织更厚

"G点不存在"侧:
  → 系统性解剖研究未发现独立于阴蒂的"G点"结构
  → 组织学检查未见独特的神经末梢集中区
  → "G点高潮"可能只是阴蒂内部结构（前庭球+阴蒂脚）从阴道壁被间接刺激

为什么是"未决":
  不是"有没有敏感区"——是"这个敏感区是不是一个独立的解剖结构"
  如果G点是阴蒂的投影（不是独立结构），那"阴道高潮vs阴蒂高潮"的二分法本身就是错的
  → 但"阴蒂是唯一的高潮器官"这个结论同样过于简化（因为阴蒂内部延伸的长度和位置因人而异）
```

### 争议二：性成瘾——真实疾病还是道德恐慌？

```
"真实疾病"侧:
  → ICD-11 包含" compulsive sexual behaviour disorder"（强迫性性行为障碍）
  → 部分人确实报告失控的性行为造成严重生活后果
  → 多巴胺失调模型——和物质成瘾有相似的神经基础

"道德恐慌"侧:
  → DSM-5 拒绝了" hypersexual disorder"的诊断——证据不足
  → "性成瘾"在公众讨论中被滥用来病理化任何高频性行为
  → 缺乏统一的诊断标准和生物标志物
  → 很大比例的"性成瘾"患者实际是：未诊断的双相障碍（躁狂期性欲亢进）+ 宗教性羞耻内化

现状: ICD-11 有"强迫性性行为障碍"但诊断要求严格（持续6月+显著功能损害+不是由其他精神疾病解释）
——这不等于"一周想要多于X次"
```

### 争议三：睾酮补充——正常值下限在哪？

```
"更低也正常"侧:
  → 衰老相关的睾酮下降是"生理性"的——不需要"纠正"
  → 大规模纵向研究：多数老年男性的睾酮在"正常偏低"范围——没有明显症状
  → TRAVERSE试验(2023): 睾酮补充未增加心血管事件（3年随访）——但获益也有限
  → 过度处方——美国睾酮处方2000-2013增加了4倍——远多于真正性腺功能减退的患病率

"应该更宽松"侧:
  → "正常范围"来自人群统计——不代表"这个范围内你就不会受益"
  → 300 ng/dL是"正常下限"——但有症状的350 ng/dL男性补充后性欲和精力改善
  → 生物可利用睾酮（游离+白蛋白结合）比总睾酮更相关——但大多只测总睾酮

未决:
  300 ng/dL这条线是统计边界，不是生理边界。但把线往下移到哪——证据不够。
```

### 争议四：色情制品——对性功能的净效应是什么？

```
"有害"侧:
  → 观察性研究：重度色情消费和ED、延迟射精、性满意度降低相关
  → "色情诱导的勃起功能障碍(PIED)"假说：逐渐需要更极端的刺激→对真实性伴侣的反应降低
  → 机制上合理：多巴胺受体下调和敏化（和物质成瘾同通路）

"无害甚至有益"侧:
  → 没有RCT证明色情会导致ED（没有伦理上可行的实验设计）
  → 相关≠因果——可能是ED导致更多色情消费（因为没有伴侣性行为）——不是反过来
  → 更多的色情消费和更好的性知识、更愿意探索性多样性相关（在部分研究中）
  → 宗教/道德反对色情的人报告更多"色情成瘾"——尽管消费量并不更高——提示道德不一致性是困扰的来源

为什么是"未决":
  横断面研究不能确定因果方向。需要纵向研究（跟踪同一群人多年，看消费量变化和性功能变化的时间顺序）。
  目前没有这样的数据。
```

### 争议五：女性高潮——"应该"通过性交达到吗？

```
"不应该要求"侧:
  → 大规模调查：只有18-30%的女性报告"仅通过阴茎-阴道性交就经常达到高潮"
  → 解剖学事实：阴蒂（非阴道）是主要的高潮器官——要求"性交高潮"就像要求"不用手吃饭"
  → Freud 的"阴道高潮成熟论"在解剖学上已被推翻——但文化上还活着

"应该帮助"侧:
  → 有女性确实报告阴道性交高潮更令人满意——并希望达到
  → 不应否定这部分女性的经验和愿望
  → 可能需要的是"调整性行为模式"（更多阴蒂刺激）——而不是"修复女性"来适应"标准性交"

调和:
  "女性的性高潮不应该以阴茎-阴道性交为唯一标准"——这个结论已经很清楚
  但说"你不应该追求某种类型的高潮"同样是越界
```

## 附录C：深度阅读路径

### 第一根：欲望
- **性欲的系统视角**: 《Come As You Are》Emily Nagoski (2015) — 双重控制模型——最易懂也最科学的性欲科普

- **NO发现史**: 1998年诺贝尔生理学奖演讲 (Furchgott, Ignarro, Murad) — NO通路的完整发现史
- **ED和心血管疾病**: 《Journal of Sexual Medicine》任意心血管-ED关联综述

### 第三根：神经
- **高潮的神经科学**: 《The Science of Orgasm》Barry Komisaruk et al. (2006) — fMRI研究+神经通路+脊髓损伤中的高潮
- **早泄**: ISSM早泄指南 (2014更新) — 诊断和治疗的国际共识

### 第四根：结构
- **盆底**: 《Evidence-Based Physical Therapy for the Pelvic Floor》Bø et al. (2015) — 不只是Kegel——盆底生物力学和康复
- **阴蒂解剖**: O'Connell et al. (2005) *The Journal of Urology* — 阴蒂内部结构的现代MRI解剖研究

### 第五根：功能障碍
- **性医学标准教材**: 《Human Sexuality and Its Problems》John Bancroft (2009) — 最全面的生物心理社会性医学教材

### 第六根：关系
- **性脚本**: 《The Social Organization of Sexuality》Laumann et al. (1994) — 芝加哥大学全国性调查——性脚本的社会学基础
- **长期关系中的性**: 《Mating in Captivity》Esther Perel (2006) — 亲密和情欲的张力——有文学性但科学基础不如上面

## 附录E：如果这本书错了——诚实面对最大弱点 ★反对声音

每本书都有盲点。以下不是"说不定"的吹毛求疵——是如果这本书的核心框架错了，最可能错在这几个地方。

### 对第一根骨头的挑战：欲望 = 加速器 - 刹车

**最有力的反对论点**：双重控制模型解释力强——但它是一个描述框架不是可证伪的理论。"加速器敏感度"和"刹车敏感度"无法被独立精确测量——它们是概念不是变量。

**什么证据会推翻它**：如果有人证明性欲变化可以完全由激素+神经递质浓度预测——不需要"刹车敏感度"这个不直接可测的概念——那么双重控制模型只是"激素模型穿了心理学术语的外套"。

### 对第二根骨头的挑战：ED = 心血管前哨信号

**最有力的反对论点**：OR 1.42 有统计显著性——但对个体预测价值有限。大多数 ED 患者不会在 5 年内发生心血管事件。

**什么证据会推翻它**：如果 ED 和 CVD 的关联完全由共享风险因素解释——ED 不是独立预测因子。

**目前的状态**：多变量校正后 OR 仍显著——支持独立预测价值。但效应量意味着临床决策不能只靠 ED。

### 对第三根骨头的挑战：高潮需要前额叶去激活

**最有力的反对论点**：fMRI 研究样本量小（N<30）、实验室情境人为性高（在 MRI 扫描仪里手淫——和真实性行为差多远）、大部分来自同一研究组。 **自然情境下的高潮脑成像数据为零。**

### 对第四根骨头的挑战：盆底训练改善性功能

**最有力的反对论点**：观察到盆底肌力↔性功能相关——但因果关系可能是反向的（更好的性功能→更多性活动→更强盆底）或第三方变量（整体健康意识）。

### 对第五根骨头的挑战：障碍分类体系的阈值

**最有力的反对论点**：DSM-5 将连续的生理/心理变化划为"有障碍"和"无障碍"——阈值是专家共识不是生物学分界。FSIAD 从 DSM-IV 到 DSM-5 改了三次名字——说明领域自己也意识到分类有问题。

### 对第六根骨头的挑战：关系系统——证据最弱的一章

**最有力的反对论点**：第六章大部分基于临床经验+横断面研究。没有 RCT 随机分配伴侣去"高冲突组"。**这一章比其他五章更接近"有见地的观点"而不是"可验证的知识"。**

---

### 整体的自我怀疑

**这本书最大的方法论局限**：性医学的证据质量和心血管/肿瘤医学不在一个量级。很多"我们知道"的陈述基于比书中引用的更弱的研究设计——因为没有更好的。RCT 在性医学中更难做（伦理、盲法、招募、资金）。

- "他们还在争论 G 点是不是独立结构——而我们已经有全盆腔神经-血管-肌筋膜交互模型了"
- "他们把睾酮当成'性欲激素'——现在我们知道性欲是肠道菌群+免疫+社会脚本的交互产物"
- "他们还在用 fMRI 看高潮——躺在扫描仪里手淫——真原始"
