# 第二章：血管系统——充血、一氧化氮与内皮

> 欲望来自大脑——但身体的准备需要血流。勃起和润滑不是"肌肉动作"——是血管事件。理解这一点，你就能理解为什么 ED 是心血管疾病的"前哨信号"。

---

## 2.0 一氧化氮——从"松弛因子"到诺贝尔奖 ★发现故事

**1980 年，纽约。** 药理学家 Robert Furchgott 在实验室里被一个奇怪的现象难住了：他给兔子的主动脉环加了乙酰胆碱——血管应该收缩还是舒张？有时候收缩，有时候舒张。最终他发现：舒张只发生在血管内皮的物理结构完好的时候——如果他不小心擦掉了内皮，血管就收缩。

**当时所有人都相信**：血管的收缩和舒张由神经末梢释放的去甲肾上腺素和乙酰胆碱直接控制。没有"内皮→平滑肌"的信号分子。

**但 Furchgott 注意到**：那个让平滑肌松弛的信号不是来自神经——是来自内皮细胞。他把它叫作 **EDRF（内皮源性松弛因子）**——一个"存在但不知道是什么"的东西。

**关键实验**：Furchgott 1980 年发表的论文描述了 EDRF 的存在。但 EDRF 到底是什么？接下来的七年里全世界都在猜。1987 年，多个实验室同时鉴定出：EDRF 就是一氧化氮（NO）——一个简单的气体分子。

**结果**：NO 是体内最不寻常的信使——不是像激素那样通过受体作用，而是直接扩散进平滑肌细胞，激活鸟苷酸环化酶→cGMP↑→平滑肌松弛→血管扩张→充血。

**然后**：1992 年，Science 杂志宣布 NO 为"年度分子"。1998 年，Furchgott、Ignarro 和 Murad 获得诺贝尔生理学奖——从第一篇论文到诺奖，18 年。

**为什么这改变了一切**：NO 通路的发现不仅解释了勃起的分子开关——它也是 PDE5 抑制剂（西地那非/他达拉非/伐地那非）开发的直接理论基础。没有 EDRF 的发现，就没有伟哥。

## 2.1 一氧化氮通路：勃起和润滑的分子开关

### 完整信号链

```
性刺激（触觉/视觉/想象）
    ↓
副交感神经末梢释放乙酰胆碱
    ↓
内皮细胞 eNOS 酶激活 → L-精氨酸 → NO
    ↓
NO 扩散进入平滑肌细胞
    ↓
激活鸟苷酸环化酶 → GTP → cGMP
    ↓
cGMP → 细胞内钙↓ → 平滑肌松弛
    ↓
螺旋动脉扩张 → 血液涌入海绵体/阴蒂/阴道壁
    ↓
充血 → 压迫静脉 → 血液被困住 → 维持勃起/充血状态
    ↓
PDE5 酶降解 cGMP → 平滑肌恢复张力 → 消退
```

**关键事实**：
- 全程依赖内皮功能——如果内皮不健康，NO 合成减少，再多的性刺激都无法触发充分的充血
- PDE5 是这个信号的"终止按钮"——PDE5 抑制剂（伟哥类）通过阻止 cGMP 降解来延缓和增强勃起。但它们不能启动勃起——没有 NO→没有 cGMP→PDE5 抑制剂没有可阻止的东西
- 女性和男性使用相同的 NO-cGMP 通路进行生殖器充血——阴蒂海绵体和阴道壁的充血依赖同一个分子开关 [S009]

## 2.2 内皮功能：血管健康和性功能的分水岭

### 内皮不只是"血管的里衬"

```
健康内皮:
  → 产生 NO（血管舒张）
  → 防止血小板聚集
  → 抑制平滑肌增生
  → 抗炎

功能不良内皮:
  → NO 产生↓（因为 eNOS 解偶联或底物不足）
  → 内皮素-1↑（强缩血管物质）
  → 活性氧↑（氧化应激破坏 NO）
  → 炎症→白细胞粘附→动脉粥样硬化开始
```

**内皮功能是可以测量的**：血流介导的血管扩张（FMD）——用超声测量肱动脉在阻断血流 5 分钟后扩张的程度。FMD 越低 = 内皮功能越差。

## 2.3 ED 是心血管疾病的"前哨信号"

### 动脉大小假说

这是理解 ED 和心血管疾病关系最关键的概念：

```
同一个人的动脉:
  冠状动脉: 直径 3-4mm
  阴茎动脉: 直径 1-2mm
  阴蒂动脉: 直径 1-2mm

同样的内皮功能下降 → 更小的动脉先出现症状
  → 阴茎/阴蒂动脉堵塞 50% → 勃起/充血障碍
  → 冠状动脉堵塞 50% → 可能还没症状（或只在运动时胸闷）
```

**2024 年 meta 分析（Borges, 39 项研究, >450,000 人）**：
- ED 患者未来发生心血管事件的 OR = 1.42 (95% CI: 1.28-1.57)
- ED 先于 CVD 诊断出现的时间窗口：**2-5 年** [S010]

**这意味着什么**：ED 不是"老了"——是一个可操作的早期预警。2-5 年的窗口期不是用来焦虑的——是用来干预的。

### 具体血管参数（Peng et al., 2024, 57 项研究, 7,312 人）

| 血管参数 | ED 患者 vs 非 ED | 含义 |
|---------|----------------|------|
| FMD | 低 2.86% | 内皮依赖性舒张功能受损 |
| 脉搏波速度 | 高 1.11 m/s | 动脉硬化 |
| 颈动脉 IMT | 高 0.11mm | 亚临床动脉粥样硬化 |
| 内皮祖细胞 | 低 0.72% | 内皮修复能力下降 |

## 2.4 PDE5 抑制剂：怎么工作、为什么不总是有效

### 三类 PDE5 抑制剂

| 药物 | 起效 | 持续 | 受食物影响 |
|------|------|------|-----------|
| 西地那非（万艾可） | 30-60 min | 4-5 h | 是（高脂餐延迟吸收） |
| 他达拉非（希爱力） | 30-120 min | **36 h** | 否 |
| 伐地那非（艾力达） | 30-60 min | 4-5 h | 是 |

### PDE5 抑制剂不总是有效的四个原因

1. **没有 NO 信号就无法工作**：性刺激→NO 是这个通路的第一步。如果缺乏性刺激（或内皮无法产生足够的 NO），PDE5 抑制剂=无用
2. **血管神经损伤**：前列腺根治术可能切断海绵体神经——即使内皮正常，神经信号到不了
3. **严重血管疾病**：如果动脉已经严重堵塞，即使平滑肌松弛了也不会有足够的血流
4. **激素缺乏**：严重睾酮低下→PDE5 表达↓→PDE5 抑制剂靶点减少

## 2.5 女性润滑和充血：同一个机制，不同的表现

### 阴道的血管充血

- 阴道壁和阴蒂海绵体使用和阴茎海绵体相同的 NO-cGMP 通路
- 性刺激→阴道血流量↑→跨壁压力→血浆滤出→阴道润滑
- 阴蒂的勃起组织（海绵体）在性刺激下充血膨大——和男性勃起机制完全相同

### 为什么 PDE5 抑制剂对女性"没用"

- 多个 RCT 测试了西地那非对女性性唤起障碍的效果——结果基本是阴性
- **可能的原因**：女性性唤起障碍的核心通常不在血流——在欲望和主观唤起（大脑）。PDE5 抑制剂解决的是"血流来了但不够"的问题——不是"大脑没有发出信号"的问题
- 但在特定人群中有阳性结果：脊髓损伤女性、SSRI 相关性功能障碍女性——即"血流问题"确实是主要障碍的人群 [S011]

## 2.6 总结：血管系统的三条铁律

1. **性功能是血管健康的反映。** ED 不是孤立事件——是内皮功能下降的第一个临床症状。一个 40 岁男性出现 ED 时，他的问题不在阴茎——在血管。

2. **2-5 年的窗口期是礼物。** ED 比心梗早 2-5 年出现——这段时间用来改善生活方式（运动、饮食、戒烟、减重），比 50 岁之后再吃他汀有用得多。

3. **PDE5 抑制剂不是"春药"。** 它们不产生欲望——只放大 NO 信号。没有性刺激→没有 NO→吃了也没用。这也解释了为什么"伟哥对女性没用"——当问题是"我不想要"而不是"我的血流不够"时，扩张血管解决不了问题。
