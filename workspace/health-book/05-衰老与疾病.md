# 第五章：衰老与疾病——身体的"故障模式"

> 前四章讲了系统怎么正常运作。这一章讲：**系统怎么出故障、以及怎么在故障发生前拦住它。**
>
> 衰老不是疾病——但衰老是所有慢性疾病的最大风险因子。理解了衰老的机制，你就理解了什么可以防、什么只能延缓、什么是基因决定的你改不了。

---

## 5.1 衰老的机制：九大标志

2013年López-Otín等人提出，2023年更新——**The Hallmarks of Aging（衰老的标志）**。这不是理论，是分子层面的可测量变化：

```
三大"原发标志"（启动衰老的）
  ① 基因组不稳定性：DNA损伤累积
  ② 端粒缩短：染色体末端的"保护套"每次细胞分裂缩短一点
  ③ 表观遗传改变：基因"开关"的错乱——不该开的开了，不该关的关了

三大"拮抗标志"（身体对原发标志的反应，但后来反噬）
  ④ 线粒体功能障碍（见第一章§1.4）
  ⑤ 细胞衰老：衰老细胞不分裂但不死——分泌炎症因子→污染邻居细胞
  ⑥ 蛋白质稳态丧失：错误折叠蛋白累积（自噬不足→见第一章§1.6）

两大"整合标志"（前六大标志的最终结果）
  ⑦ 干细胞耗竭：组织再生能力耗尽
  ⑧ 细胞间通讯改变：炎症信号过多、修复信号过少
  ⑨ 巨自噬受损（2023年新增）
```

**理解这九大标志的意义**：衰老不是一种病——它是多种病的共同土壤。抗衰老不是"找一颗不老药"——是同时干预多个标志。

## 5.2 四大骑士之一：心血管疾病

### 不是"胆固醇高"这么简单

```
动脉粥样硬化的全过程：
① 血管内皮损伤（高血压/高血糖/吸烟/炎症→内壁划痕）
② LDL颗粒进入内皮下
③ 氧化→被巨噬细胞吞噬→泡沫细胞→脂肪条纹
④ 平滑肌迁移+胶原沉积→纤维帽形成→斑块
⑤ 斑块破裂→血栓形成→心梗/中风
```

### ApoB 和 Lp(a) ——比 LDL-C 更重要的指标

**传统体检只查 LDL-C（低密度脂蛋白胆固醇浓度）。但 LDL-C 不是"坏胆固醇颗粒的个数"——它是浓度。一个 LDL 颗粒可以大而轻（相对无害）或小而密（高度致动脉粥样硬化）。LDL-C 不能区分这两种。**

**ApoB 比 LDL-C 更准确。** 每个致动脉粥样颗粒（LDL+VLDL+IDL+Lp(a)）表面恰好有一个 ApoB 分子。测 ApoB = 直接数"有多少致动脉粥样颗粒"。2026年 ACC/AHA 指南已给出 Class 2a 推荐：在 ASCVD/CKM综合征/T2D/高TG患者中，测 ApoB 来指导治疗强化 [H005]

**Lp(a) 是更重要的独立风险因子，而且是遗传性的。** Lp(a) 水平 80-90% 由基因决定——饮食和运动几乎不能改变它。EAS 2022共识推荐每个成年人一生至少测一次Lp(a)。风险以7 mg/dL为参照（白人中位）：30→1.22倍，50→1.40倍，100→1.95倍——风险沿连续梯度递增，非单一阈值。各指南阈值不同（EAS用分层/NLA用50mg/中国用30mg）。[H003]

**中国人群特别警惕：** 2025 年 JACC 发表的一项 290 万中国成人的研究显示 Lp(a) 升高在中国人群中存在显著负担。中国专家共识推荐 **30 mg/dL** 为风险阈值——比国际通行的 50 mg/dL 更低，因为中国人群 Lp(a) 在更低水平即出现风险等效 [H004]

### Peter Attia 的"心血管战争"框架

```
防线1（一级预防）——在你还没有任何症状时：
  → 测ApoB和Lp(a)——不是只测LDL-C
  → 将ApoB降至目标水平（按风险分层）
  → 控制血压（目标<120/80）
  → 不吸烟、运动、保持代谢健康

防线2（早期干预）——在斑块已经存在但还小的时候：
  → 冠脉钙化积分(CAC)扫描——40+男性、50+女性应至少做一次
  → CAC>0 → 你已经有动脉粥样硬化了 → 立即开始降脂治疗
  → 继续运动+饮食，但此时单独生活方式可能不够

防线3（二级预防）——已经发生过心血管事件：
  → 更激进的ApoB目标
  → 抗血小板药物
  → 严格控制所有风险因子
```

**Attia 最核心的观点**：心血管疾病是一种"儿科疾病"在老年才显现——斑块在十几岁时就开始形成。你40岁体检报告"胆固醇正常"不代表你安全——可能只是斑块还在增长还没破。"正常"是统计概念（在人群分布的非异常范围内），不是你的个人最佳水平。

## 5.3 四大骑士之二：癌症

### 癌症不是一种病——是几百种病的统称

共同特征：**逃避细胞凋亡的失控分裂**

```
癌症的发生（多击模型）：
  ① 原癌基因激活（基因突变"踩油门"→持续生长信号）
  ② 抑癌基因失活（p53突变→"刹车"失灵）
  ③ 端粒维持（癌细胞激活端粒酶→绕过分裂次数限制）
  ④ 血管生成（肿瘤长出自己的血液供应）
  ⑤ 免疫逃逸（躲避免疫系统的识别和清除）
  ⑥ 转移（侵入其他组织→致命）
```

**这意味着**：
- "治愈癌症"像"治愈所有病毒感染"——每个癌症有不同的分子机制
- 早期发现的价值远大于晚期治疗——**第一颗肿瘤细胞和"体检摸到的肿块"之间可能隔了10-20年**
- 免疫力是最被低估的防线——你的免疫系统每天清除数百个癌变细胞。免疫逃逸是癌症的最后一步

### 预防策略的现状

| 癌症类型 | 有效筛查 | 可干预风险因子 |
|---------|---------|-------------|
| 肺癌 | LDCT（低剂量CT） | 吸烟（占肺癌死亡85%+） |
| 结直肠癌 | 结肠镜（45岁起） | 加工肉（WHO 1类致癌物）、肥胖、酒精、纤维不足 |
| 乳腺癌 | 乳腺X线（40+） | 肥胖（绝经后）、酒精、缺乏运动 |
| 宫颈癌 | HPV检测+涂片 | HPV疫苗（近乎100%预防） |
| 肝癌 | 超声+AFP（高风险人群） | 乙肝疫苗、控制脂肪肝、限酒 |

**癌症预防的最大变量**：不吸烟、不喝酒、保持体重、运动。四项加起来可以预防约40%的癌症。剩下的60%是基因+环境+随机突变——这不是悲观，是诚实。

## 5.4 四大骑士之三：神经退行性疾病

### 阿尔茨海默 = "3型糖尿病"？

**首先要诚实：这是一个假说（de la Monte 2008年提出），不是被正式接受的医学分类。** 但它是现在神经科学最活跃的研究方向之一。AD患者脑组织中确实存在胰岛素抵抗（Talbot 2012, JCI确认），但因果关系方向未定——脑胰岛素抵抗可能是AD的原因，也可能是AD的结果。Aβ→TNF-α/JNK→IRS-1的通路在体外和动物模型中已验证，但临床转化证据有限。这是多种共存假说（炎症假说、胆碱能假说、tau假说、血管假说）之一。**[H006]**

```
脑胰岛素抵抗 → 阿尔茨海默的传导链：
  ① 外周胰岛素抵抗 → 慢性高胰岛素血症 → 胰岛素穿过血脑屏障减少
  ② 脑内胰岛素↓ → 神经元葡萄糖摄取↓ → 能量危机
  ③ 能量不足 → Aβ前体蛋白异常剪切 → Aβ42寡聚体形成
  ④ Aβ寡聚体 → TNF-α/JNK通路 → IRS-1丝氨酸磷酸化 → 脑胰岛素受体功能障碍
  ⑤ GSK-3β过度活跃（胰岛素正常时应抑制它）→ tau过度磷酸化 → 神经纤维缠结
  ⑥ 氧化应激+炎症+突触损伤 → 神经元死亡 → 认知功能下降
```

**从假说到证据**：
- 2005年 Steen 首次提出"AD是3型糖尿病"
- AD 患者脑部尸检显示胰岛素信号通路全部受损（与 T2D 患者外周组织一样的分子模式）
- 动物实验：激活 PPAR-δ→改善脑胰岛素敏感度→减少 tau 磷酸化和 Aβ 积累
- 2024年 Ornish 等的 RCT（n=51, 20周）：强化生活方式干预对MCI/早期AD有初步阳性信号——但样本极小、随访短、回顾性注册(2017开始→2020才注册)、多重干预无法分离有效成分。需要更大样本独立重复。[H007]

**这意味着什么？** 管好代谢——特别是胰岛素敏感度——可能是预防阿尔茨海默最有效的手段。这不是"吃某种超级食物防痴呆"——是"你现在的代谢健康决定了你30年后的脑健康"。

## 5.5 四大骑士之四：2型糖尿病与代谢综合征

### 为什么说它是"万病之源"

```
代谢综合征的诊断标准（满足3条即诊断）：
  □ 腰围：男≥90cm，女≥85cm（中国标准）
  □ 空腹血糖≥5.6 mmol/L
  □ 血压≥130/85 mmHg（或正在服药）
  □ 甘油三酯≥1.7 mmol/L
  □ HDL-C：男<1.04，女<1.30 mmol/L
```

代谢综合征的核心是胰岛素抵抗——所以它和第一章的代谢灵活性是一体两面。代谢综合征患者的全死因死亡率升高约 1.5 倍，心血管死亡率升高约 2 倍。

### 中国数据

中国成人糖尿病患病率~12.8%（ADA 标准），前期~38.1%。这意味着**超过一半的中国成人要么已经确诊糖尿病，要么在去确诊的路上。**

更惊人的是：中国有约 30% 的成人患有 NAFLD（非酒精性脂肪肝），其中很多人不知道。NAFLD 是胰岛素抵抗的肝脏表现——把它理解为"肝脏的糖尿病前状态"。

## 5.6 衰老可以干预吗？Sinclair vs Attia vs 功能医学

### 三派之争

```
Sinclair（Lifespan, 2019）：信息论
  → 衰老的核心机制是表观遗传信息随时间丢失
  → "衰老是一种病，而且可以治疗"
  → 工具：NMN/NR(补NAD⁺)+Sirtuins+重编程
  → 激进乐观——相信在可预见的未来人类可以活到120-150

Attia（Outlive, 2023）：战术论
  → "我不关心你能活到多少岁——我关心你80岁时能做什么"
  → "百岁十项全能"：反推设计——你80岁想做什么→60岁该训练什么→40岁该准备什么
  → Medicine 3.0哲学：从被动治疗→主动预防。不是"不去医院"——是"在去医院的15年前就开始干预"

功能医学：系统论
  → 慢性病是多因素相互作用的结果
  → 工具：全套生物标志物+肠道检测+基因检测+荷尔蒙评估
  → 重点在"找到每个人的独特失衡"
```

### 三派的交叠共识

```
Sinclair说         Attia说              功能医学说
─────────────────────────────────────────────────
补NAD⁺          →  运动是最好的NAD⁺提升   →  +营养素支持
间歇断食         →  限时进食             →  +个性化调整
冷暴露           →  "不错但优先级低"     →  +结合呼吸训练
运动             →  最重要的长寿"药"     →  +适当类型和强度
```

**Attia 的观点对我的框架最有用**：不是因为他的结论最对——是因为他的方法论最清晰。"百岁十项全能"不是让你买他的计划，是让你自己思考——"我80岁想能做什么？然后从现在开始训练什么来确保那一天到来。"

## 5.7 长寿药：什么是真的、什么是炒作

### NAD⁺前体 (NMN/NR)

**人体能确认的**：短期（<1年）安全（最高测试NR 2000mg/天，NMN 1200mg/天）、能提升血液NAD⁺水平、适度改善胰岛素敏感度。**不能确认的**：长期安全性（无>1年数据）、人类寿命延长。**需要警惕的**：FDA 2022年已将NMN从膳食补充剂中移除；癌症安全性存在双向矛盾——可能抑制初始肿瘤也可能滋养已有肿瘤(Palmer 2021)；提升NAD⁺的机制依赖肠道菌群转化(Christen 2025 Nature Metabolism)——个体效果差异大。[H008]

### 雷帕霉素

**小鼠**：延长中位寿命 9-26%（ITP项目，特定品系/剂量）。**人**：PEARL试验(2024, n=115, 48周)——主要终点**阴性**。次要终点中女性瘦体重有改善趋势（未达统计学显著），总体不证明临床意义的健康span改善。Elicit系统分析（500篇筛选，最严格标准）找到零篇合格RCT直接检验雷帕霉素在健康老年人中的抗衰老效果。每周5-10mg是长寿方案常用剂量，但长期安全性数据仍在积累。[H009]

### 二甲双胍

流行病学数据有争议——Bannister(2014)发现二甲双胍T2D使用者比非糖尿病人群死亡率更低，但丹麦Keys(2022)**未能重复**这一发现，且Bannister研究存在显著的immortal time bias。TAME试验（3000人,14中心,6年RCT）自2015年FDA获批后持续面临资金延迟，目前仍在筹款阶段。非糖尿病人群中二甲双胍的对照试验结果为mixed–negative，且可能干扰力量训练适应性。[H010]

### 底线

**2026 年还没有任何药物被证明可以延长人类寿命，但有足够证据支持特定生活方式（运动+营养+睡眠）可以延长健康寿命。** 如果你在做"我该不该吃 NMN"的决策——先确认你已经把运动做到位了。运动增加的 NAD⁺远超过任何补充剂——而且是免费的。

## 5.8 总结：衰老与疾病的五条铁律

1. **衰老不是一种病——是多种病的共同土壤。** 九大标志相互关联——针对单个标志效果有限，综合干预（运动+营养+睡眠+压力管理）才可能延缓整个衰老过程。

2. **心血管风险不能用"胆固醇正常"来排除。** 测 ApoB 和 Lp(a)——一次就好。Lp(a) 是遗传性的、一生基本不变。如果高，就需要更激进地管理其他风险因子。

3. **阿尔茨海默在大脑，根源可能在代谢。** "3型糖尿病"是一个值得重视的假说，但尚未被正式证实。脑胰岛素抵抗和AD之间存在关联（这一点已确认），但因果方向未定。不论因果——保护大脑的最有效策略，可能就是你现在为代谢健康做的那些事情：运动、睡眠、饮食、管理胰岛素。预防窗口在症状出现前15-20年。

4. **代谢综合征不是"老了自然会得"——是累积伤害的量化。** 中国超过一半成人已经在糖尿病谱系上。好消息是：生活方式干预（DPP 试验）可以将进展为糖尿病的风险降低 58%——比二甲双胍（31%）更有效。

5. **长寿药还在实验室。长寿的生活方式现在就能做。** 运动是最好的促长寿"药物"——改善每一个衰老标志，效应量超过目前任何药物。VO₂max 是最强长寿预测指标。每增加 1-MET → 死亡率降 11%。[H001][H002]

**📝 关于衰老研究的科学诚实性：** Sinclair的信息衰老论面临重大批评——Brenner(2024, Cell)指出"信息论尚未被检验"。白藜芦醇作为sirtuin激活剂已于2005年被证伪（荧光探针非特异性结合），但Sinclair仍为其辩护。2024年Sinclair因狗补充剂争议辞去衰老研究学会主席。**部分重编程**(OSK因子,去除c-Myc)在小鼠中逆转了多组织年龄相关变化——但安全窗口极窄(仅雄性小鼠、缺少AAV对照)。论文发表在Cellular Reprogramming(非Cell)。Life Biosciences ER-100预计2026年进入人类眼部临床试验。肿瘤安全性仍为临床转化首要障碍(OSK本身保留多能性潜力)。这是最前沿但也是最受争议的方向。[H014][H015]
