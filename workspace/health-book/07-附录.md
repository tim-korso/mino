## 附录B：争议图鉴——我们不知道什么 ★诚实标记

科学最诚实的话不是"这是对的"——是"这个我们还不知道"。以下是健康领域最大的未决争议。每一条都有两边的高质量证据——不是"负面证据"，是"真不知道"。

### 争议一：低碳水 vs 低脂——宏量营养素最优比例到底是什么？

```
低碳水侧:
  → 短期(6-12月)减重效果更优
  → 改善甘油三酯和HDL-C
  → 对胰岛素抵抗/T2D患者改善血糖控制
  → 机制理论上合理: 胰岛素↓→脂肪氧化↑

低脂/地中海侧:
  → 长期(>12月)依从率更高(碳水被限制后大多数人反弹)
  → PREDIMED(地中海饮食+橄榄油/坚果): CVD一级预防RCT证据——低碳水没有同等级证据
  → 极低碳水(<50g/天)长期安全性数据缺乏
  → 地中海饮食的多项长期RCT硬终点数据更扎实

为什么是"未决"而不是"一方赢了":
  短期代谢改善(低碳水) vs 长期行为可持续性+硬终点数据(地中海)
  → 同一个人的"最佳"可能随年龄和目标变化
  → 30岁想减脂→低碳水可能更好
  → 60岁想防心梗→地中海可能更好
  → "最优"可能取决于个体: 肠道菌群+胰岛素敏感度+基因+偏好
```

### 争议二：卡路里 vs 激素——胖是因为"吃多了"还是"胰岛素太高"？

```
"卡路里进卡路里出"(传统模型):
  → 热力学第一定律——无法被违反
  → 任何饮食减重都因为热量赤字(即使是低碳水也最终减了热量)
  → NIH 2019代谢病房研究: 超加工饮食多吃500千卡→直接证明了热量盈余=长胖

"碳水化合物-胰岛素模型"(激素模型):
  → 为什么要用"热量"而不是"什么驱动了热量摄入"来解释？
  → 胰岛素↑→脂肪储存↑→循环燃料↓→饥饿感↑→想吃更多→不是意志力问题
  → 解释了为什么低碳水组在自由进食条件下自然减少热量摄入(不被要求少吃)

调和:
  两种模型在"减重成功需要热量赤字"上一致——不一致在"怎么最容易达到热量赤字"。
  卡路里模型说: 少吃就行。激素模型说: 改变吃的东西→激素模式改变→自然少吃→更容易。
  → 这不是"对vs错"——是"终极原因vs直接原因"
```

### 争议三：他汀一级预防——胆固醇不高的人该不该吃？

```
"应该吃"侧:
  → 大量RCT证据: 他汀降低LDL-C→降低心血管事件
  → "胆固醇正常"不等于"没有斑块"——ApoB和Lp(a)可能仍然高
  → 绝对风险降低: 在正确的人群中，他汀预防1例心血管事件需要治疗~50人5年

"不该普遍吃"侧:
  → 相对风险降低看起来大(30-50%)但绝对风险降低小(2-3%)
  → 副作用: 肌肉疼痛(5-10%)、新发糖尿病风险轻微增加、认知模糊(争议)
  → 不能简单将二级预防的绝对收益外推到一级预防
  → "如果LDL-C不高但Lp(a)高→用PCSK9抑制剂更有针对性——但贵"
```

### 争议四：NAD+前体——补NMN/NR有用吗？

```
"有用"侧:
  → 衰老→NAD+水平↓→Sirtuins无法工作→理论上补NAD+前体→恢复长寿基因功能
  → 动物实验: NMN改善小鼠胰岛素敏感度、线粒体功能、运动耐量
  → 人体: 短期(6-12周)提升血液NAD+水平、适度代谢改善

"没用的"侧:
  → 人类寿命延长=零证据
  → FDA已将NMN从膳食补充剂移除(2022)
  → 癌症安全性: NAD+可能滋养已有肿瘤
  → 提升NAD+依赖肠道菌群转化→个体效果差异巨大
  → Peter Attia和Nir Barzilai都不推荐普通人吃(证据不足)

现状: 对普通人来说，把钱和精力花在运动+睡眠+蛋白质上——ROI远高于NMN。
```

### 争议五：断食——不吃早餐到底好不好？

```
"断食好"侧:
  → TRE(时间限制进食): 改善胰岛素敏感度+代谢灵活性
  → 自噬在禁食>14小时被激活
  → 人类进化: 狩猎采集者每天吃2-3次(非6-7次)，基因对间歇性食物短缺更适应

"吃早餐好"侧:
  → 观察性研究: 吃早餐者BMI更低——但这是关联不是因果(吃早餐的人整体生活习惯更好)
  → 清晨皮质醇最高→胰岛素敏感度最好→此时吃碳水对血糖冲击最小
  → 运动员/高强度工作者: 晨训前不吃=表现差

调和:
  "早餐是一天中最重要的一餐"没有科学证据——是食品工业营销
  "断食到午餐"对代谢健康可能有益——但对特定人群(训练/工作需求)不一定适合
  → 回答是"取决于你的目标、日程和身体反应"——不是"吃"或"不吃"的二选一
```

## 附录C：深度阅读路径

每根骨头的进阶读物——不是"必读100本"，是你按需深入。

### 第一根：能量与代谢
- **为什么卡路里不是你想象的那样**: 《Burn》Herman Pontzer (代谢率的真相——中年代谢不会自然变慢)
- **胰岛素**: 《The Diabetes Code》Jason Fung (胰岛素抵抗+断食——有争议但可读)

### 第二根：营养与摄入
- **为什么营养建议总变**: 《The China Study》T. Colin Campbell (全面的植物证据，但激进) + 注意H019关于FFQ的方法论限制
- **个体化营养**: 《Glucose Revolution》Jessie Inchauspe (血糖管理的实操指南)

### 第三根：运动与结构
- **VO₂max和百岁十项全能**: 《Outlive》Peter Attia (本书主要参考——重点读运动+稳定性章节)
- **力量训练的为什么**: 《Starting Strength》Mark Rippetoe (入门到进阶的全面力量指南)

### 第四根：睡眠与修复
- **睡眠科学**: 《Why We Sleep》Matthew Walker (全面的睡眠科学，注意H016/H017中的方法论限制)
- **昼夜节律**: 《The Circadian Code》Satchin Panda (TRE+昼夜节律)

### 第五根：衰老与疾病
- **长寿科学全景**: 《Lifespan》David Sinclair (注意H015——Sinclair的乐观有争议)
- **胆固醇真相**: 《The Great Cholesterol Myth》Jonny Bowden (有偏向，但解释了ApoB/Lp(a)/炎症的三合一框架)

### 第六根：整合运用
- **Blue Zones**: 《The Blue Zones》Dan Buettner
- **习惯形成**: 《Atomic Habits》James Clear (不是健康书——但对"怎么把健康行为变成习惯"最有实操价值)
