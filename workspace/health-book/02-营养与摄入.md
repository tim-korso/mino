# 第二章：营养与摄入——身体的"原材料"

> 第一章讲了能量怎么被处理。现在是：往里倒什么？
>
> 营养学是健康领域最吵的地方——生酮 vs 低脂、素食 vs 肉食、间歇断食 vs 少食多餐。每一派都有研究支持、每一派都有网红代言。
>
> 这一章不告诉你"吃哪种饮食"——它告诉你：**宏量和微量营养素各自干什么、不同饮食模式的证据金字塔长什么样、以及为什么同一种饮食对不同人效果完全不同。**

---

## 2.1 宏量营养素：三项全能

### 碳水——不是敌人，是燃料

```
碳水化合物的分类：
┌────────────────────────────────────────────┐
│ 简单碳水（糖）                                │
│   葡萄糖、果糖、蔗糖（葡萄糖+果糖）              │
│   → 快速吸收 → 血糖快速↑ → 胰岛素快速↑        │
│   来源：水果、蜂蜜、白砂糖、含糖饮料            │
│                                              │
│ 复杂碳水（淀粉+纤维）                          │
│   直链淀粉、支链淀粉、可溶/不可溶纤维           │
│   → 消化较慢 → 血糖缓升 → 胰岛素缓升           │
│   来源：全谷物、薯类、豆类、蔬菜               │
│                                              │
│ 超加工碳水（精制+添加糖+脂肪的混合）             │
│   → 快速吸收 + 多巴胺爆发 + 几乎不会触发饱腹感   │
│   来源：薯片、饼干、甜甜圈、大多数零食           │
└────────────────────────────────────────────┘
```

**碳水的核心问题不是"吃不吃"，是"吃哪种"和"吃多少"。** 全谷物+薯类+蔬菜的碳水和含糖饮料的碳水，在你体内的代谢轨迹完全不同。

### 蛋白质——最被低估的宏量营养素

```
蛋白质的功能（不止"长肌肉"）：
  → 肌肉合成（最知名）
  → 免疫球蛋白（抗体的原材料）
  → 酶的组成（消化、代谢都需要酶）
  → 激素的原材料（胰岛素就是一种蛋白质）
  → 运输蛋白（血红蛋白运氧气）
  → 结构蛋白（胶原蛋白维持皮肤、骨骼、关节）
```

**为什么蛋白质比你想的重要**：

1. **饱腹感最强。** 等热量下，蛋白质提供的饱腹感远高于碳水和脂肪。这是为什么"高蛋白饮食"对减重有效——不是因为它有什么魔法，是因为吃了蛋白质你自然少吃了别的东西。

2. **热效应最高。** 消化蛋白质消耗约20-30%的自身热量（碳水5-10%，脂肪0-3%）。吃100千卡蛋白质，净得70-80千卡。

3. **防止肌肉流失。** 减重时最大的悲剧是"减掉的体重一半是肌肉"。足够蛋白质（每公斤体重1.2-1.6克）可以显著减少减重中的肌肉流失。

4. **30克阈值。** 单餐约30克蛋白质才能有效触发肌肉蛋白质合成。把蛋白质分散到三餐比集中在一餐更有效。

### 脂肪——最被误解的宏量营养素

```
脂肪的分类：
┌────────────────────────────────────────────┐
│ 饱和脂肪（室温下固体）                         │
│   → 动物脂肪、椰子油、黄油                     │
│   → LDL-C↑——但不同食物来源的饱和脂肪效果不同    │
│                                              │
│ 不饱和脂肪（室温下液体）                        │
│   → 单不饱和：橄榄油、牛油果、坚果              │
│     → 降低LDL-C，不影响HDL-C                   │
│   → 多不饱和：鱼油(Omega-3)、植物油(Omega-6)    │
│     → Omega-3抗炎，Omega-6促炎（但要平衡）      │
│                                              │
│ 反式脂肪（工业氢化）                           │
│   → 人造黄油、油炸食品                         │
│   → LDL-C↑ + HDL-C↓ + 炎症↑ = 三重打击         │
│   → WHO建议完全戒除                            │
└────────────────────────────────────────────┘
```

**"脂肪让你胖"是过去40年营养学最大的误导。** 脂肪不直接让你胖——胰岛素抵抗和持续热量盈余让你胖。事实上，足够的不饱和脂肪对激素合成、细胞膜完整性和脂溶性维生素吸收是必须的。

## 2.2 微量营养素：不缺就不用补

### 维生素和矿物质的真实作用

不是"越多越好"。微量意味着"需要量极少"。

```
最容易缺乏的四种微量营养素（中国人群）：

① 维生素D
   → 来源：日晒（主要）、脂肪鱼、蛋黄
   → 中国成人缺乏率：~50-70%（25(OH)D < 20 ng/mL）
   → 但补充剂证据矛盾——VITAL试验(n=25,871)未发现补充维D降低2型糖尿病发病率 [H012]

② 铁
   → 来源：红肉、肝脏、菠菜（非血红素铁吸收率低）
   → 中国育龄女性缺铁性贫血率：~20%
   → 缺铁→血红蛋白↓→携氧能力↓→疲劳、注意力差

③ 维生素B12
   → 来源：几乎只在动物性食物中
   → 严格素食者必须补充
   → 缺乏→巨幼细胞性贫血+神经损伤（不可逆）

④ 镁
   → 来源：绿叶蔬菜、坚果、全谷物
   → 参与300+酶反应——从ATP合成到DNA修复
   → 可能增加维生素D合成菌 [H013]
```

**关键原则**：缺了就补，不缺不补。市面上的"综合维生素"对饮食均衡的人来说，在大多数大型RCT中未显示明确的疾病预防效果。

## 2.3 饮食模式对比——证据金字塔

### 不是哪种"赢了"——是每种有不同的适用场景

| 饮食模式 | 核心特征 | 最强证据 | 弱点 |
|---------|---------|---------|------|
| **地中海饮食** | 橄榄油+蔬菜+鱼+全谷物+少量红酒 | 🟢 CVD一级/二级预防(PREDIMED试验) | 无 |
| **DASH饮食** | 低钠+高钾/镁/钙+全谷物+蔬果 | 🟢 降血压(RCT证据最强) | 无 |
| **低碳水/生酮** | 碳水<10-20%总热量，高脂肪 | 🟡 短期减重+癫痫治疗+代谢综合征改善 | 长期安全性数据有限，依从率低 |
| **间歇断食** | 限时进食，不限制食物种类 | 🟡 减重+胰岛素敏感度改善 | 长期坚持率存疑 |
| **植物性饮食** | 全食物植物为主，低动物蛋白 | 🟢 China Study+Adventist Health Studies→CVD/癌症↓ | B12/铁/蛋白质不足风险 |
| **中国居民膳食指南** | 谷类为主+蔬果+适量肉蛋奶 | 🟢 中国人群数据+低营养不良率 | 对中国慢性病预防效果次于地中海（证据体量较小） |

**最重要的结论**：上面这六种饮食的共同点远大于分歧——都强调全食物、蔬菜、限制超加工食品和添加糖。分歧主要在于宏量营养素比例，而这个比例对健康结局的影响远小于"吃的是真食物还是工业产品"。

**📝 关于蛋白质来源的最新证据**：用植物蛋白替代红肉/加工肉→CVD死亡率和全因死亡率显著降低(Harvard 2024,30年随访)。但"动物蛋白"不是单一类别——鱼类和乳制品的替代效益不明显或呈中性。**利益最明确的是"减红肉/加工肉"，不是"所有动物蛋白都有害"。** 这与Campbell(2005)的激进结论不同——营养科学在细化。[H018]

**📝 营养建议为什么总变？** 营养流行病学的根本方法论问题：FFQ(食物频率问卷)设计用于"排序"非"量化"。而替代模型(substitution modelling)需要精确的绝对量——这是FFQ无法提供的。系统误差主导FFQ的测量误差结构，标准统计校正方法不足(Louie & Bhowmik, Nature EJCN 2026)。当你看到"吃X增加Y%的Z风险"的头条时——先看是观察性研究还是RCT。如果是观察性研究且用FFQ——效应量可能被系统误差严重扭曲。[H019]

## 2.4 肠道菌群——你的"第二大脑"

### ~40万亿细菌在你体内

```
肠道菌群的功能：
  → 消化：分解膳食纤维→短链脂肪酸(SCFA)→给肠道细胞供能
  → 免疫：训练免疫系统区分"敌友"→免疫耐受
  → 代谢：影响能量吸收效率、调节胰岛素敏感度
  → 神经：通过肠-脑轴影响情绪和食欲（迷走神经+血清素90%在肠道产生）
  → 维生素合成：维生素K和B族维生素的部分来源
```

### 菌群多样性 = 代谢健康

- 多样性越高 → 代谢越灵活 → 肥胖/糖尿病风险越低
- 多样性越低 → 典型西方饮食（低纤维+高加工食品）→ 菌群"沙漠化"

**怎么养好你的菌群**：
1. **吃纤维（每天25-30克）**——这是菌群的食物
2. **吃发酵食品**——酸奶、泡菜、纳豆（活的益生菌来源）
3. **多样化蔬果**——每周30+种不同植物食物→菌群多样性↑
4. **避免不必要的抗生素**——一次广谱抗生素可以破坏菌群数月
5. **运动**——独立于饮食，运动本身可以改善菌群组成

## 2.5 超加工食品：为什么你停不下来

### 不是你的意志力有问题——是食物被设计成"停不下来"

```
超加工食品的特征：
  → 高热量密度 + 低营养密度
  → 精确调制的"极乐点"（盐/糖/脂肪的最佳比例）
  → 几乎不需要咀嚼 → 胃的膨胀信号来不及触发 → 吃多了
  → 快速吸收 → 血糖过山车 → 2小时后又饿了
  → 多巴胺爆发 → 和成瘾物质激活同一脑区
```

**NIH 2019年的里程碑试验**：让20个成年人住进代谢病房，随机分配超加工饮食 vs 全食物饮食（热量、宏量营养素、纤维、盐全部匹配）。唯一的区别是"有没有被加工过"。结果：超加工饮食组每天多吃**508千卡**，两周后体重增加**0.9公斤**——而全食物组体重下降了0.9公斤。

**这不是热量计算的问题——是食欲调控系统被超加工食品劫持了。**

## 2.6 个体化营养：为什么同一种饮食对你和我不一样

### 血糖反应——双胞胎的启示

2015年以色列Weizmann研究所的里程碑研究：给800人吃同样的食物，测餐后血糖反应。结果：**同样吃一根香蕉，有人血糖飙到140，有人几乎不变。**

预测因素（按重要性排序）：
1. 肠道菌群组成（最重要的单一预测因子）
2. 睡眠质量（前一晚睡得差→明天碳水反应差）
3. 进食顺序（先吃蔬菜/蛋白质→再吃碳水→血糖峰值降30-40%）
4. 餐后运动（饭后散步15分钟→血糖峰值降25%）

**实用意义**："碳水应该占多少"这个问题的答案对你的邻居和对你可能完全不同。唯一的办法是自我实验——连续血糖监测(CGM)+记录食物和感受。血糖反应的差异大到可以让一种食物从"健康"变成"致胖"——仅仅因为个体差异。

## 2.7 总结：营养的三条铁律

1. **宏量营养素没有"好坏"——要看来源和加工深度。** 碳水有好坏（燕麦 vs 薯片），脂肪有好坏（橄榄油 vs 反式脂肪），蛋白质有好坏（鱼 vs 加工肉肠）。最关键的特征不是"是哪种宏量营养素"，是"离农田有多远"。

2. **肠道菌群是营养学的"暗物质"。** 它解释了为什么同一种饮食对不同人效果完全不同。保护菌群多样性（纤维+发酵食品+避免不必要的抗生素）可能比你选择哪种具体饮食模式更重要。

3. **超加工食品是唯一真正"坏"的食物。** 不是因为它们的宏量营养素——是因为它们和正常食物在食欲调控系统上的效果完全不同。不是它们让你意志力差，是它们被设计成绕过你的饱腹信号。
