# 第一章：能量与代谢——身体的"货币体系"

> 金融书的第一章讲钱从哪来。健康书的第一章讲能量从哪来、怎么被处理。
>
> 不管你吃什么、怎么动、睡多久——最终都要回答一个问题：**身体怎么处理能量？**
>
> 这一章回答：ATP是什么、代谢率为什么每个人不一样、胰岛素为什么是总开关、线粒体怎么烧燃料、为什么"少吃"不一定瘦、以及断食在细胞层面做了什么。

---

## 1.1 热量的本质：ATP——你的一切活动都在烧这个

### 不是"卡路里进、卡路里出"那么简单

这句话技术上没错——热力学第一定律——但它就像说"穷是因为花的比挣的多"。不错，但没有用。真正的问题是：**为什么花的比挣的多？**

答案藏在 ATP 里。

### ATP：身体的能量货币

```
你吃的食物（碳水/脂肪/蛋白质）
    ↓ 消化
葡萄糖 + 脂肪酸 + 氨基酸
    ↓ 细胞呼吸（线粒体）
ATP（三磷酸腺苷）
    ↓ 水解（ATP → ADP + Pi）
能量释放 → 肌肉收缩、神经传导、蛋白质合成、维持体温
```

**ATP 是唯一能被细胞直接使用的能量形式。** 你吃的所有东西最终都要被转化成 ATP 才能用。你不是"吃卡路里"——你是吃原料，让线粒体帮你转换成 ATP。

一个成年人每天产生和消耗的 ATP 大约等于自己的体重。换句话说，你身体的每个 ATP 分子每天被回收再利用几百到上千次。

### 三条能量系统——不同时间尺度的"燃料"

```
时间尺度           系统              燃料              产物
─────────────────────────────────────────────────────────
0-10秒           磷酸肌酸系统       磷酸肌酸           ATP
10秒-2分钟        无氧糖酵解         葡萄糖             乳酸 + ATP
2分钟+            有氧氧化          葡萄糖/脂肪酸       CO₂ + H₂O + 大量ATP
```

**为什么这重要？** 因为它解释了为什么"慢跑减肥、冲刺不减"——冲刺用的是磷酸肌酸和糖酵解（效率低、不烧脂肪），慢跑用的是有氧氧化（效率高、直接烧脂肪）。但也解释了为什么"只慢跑不练力量代谢率一直低"——力量训练让磷酸肌酸系统效率提升→肌肉量增加→静息代谢率提高。

## 1.2 代谢率的真相

### 基础代谢率 ≠ 你以为的那样

```
总能量消耗的构成：
┌────────────────────────────────────────┐
│ 基础代谢率（BMR）          ~60-70%     │  ← 维持生命活动（心跳/呼吸/体温/细胞修复）
│ 食物热效应（TEF）          ~10%        │  ← 消化食物本身消耗的能量
│ 非运动活动产热（NEAT）     ~15%        │  ← 走路、站立、做家务、抖腿
│ 运动消耗（EAT）            ~5-10%      │  ← 健身房、跑步、游泳
└────────────────────────────────────────┘
```

**三个你不知道的事实**：

**第一，人到中年代谢不会自然变慢。** 燃（Burn, 2022）这本书的核心发现：20-60岁之间的代谢率基本恒定——如果你胖了，不是因为年龄让你代谢变慢，是因为你动得更少、吃得更多。真正代谢下降从60岁以后才开始。

**第二，NEAT 是最大的变量。** 两个同样体重、BMR相近的人，一个每天消耗2000千卡，另一个2500千卡。那多出的500千卡通常不在健身房——在那个人更多地走路、站立、做家务。"不爱动"和"爱动"之间的NEAT差距可以高达每天350-2000千卡。

**第三，运动不是减肥的主要途径。** 一小时跑步消耗约500千卡，一个汉堡就补回来了。运动的健康价值远大于减肥价值——它改善的是代谢质量（胰岛素敏感度、心肺功能、肌肉量），不是代谢数量。

## 1.3 胰岛素的中心地位

### 胰岛素不是"坏东西"——它是总开关

如果健康知识只能讲一件事，就讲胰岛素。

```
进餐 → 血糖↑ → 胰腺β细胞释放胰岛素
    │
    ├── 肝脏：停止产糖、开始储存糖原
    ├── 肌肉：吸收葡萄糖、储存糖原  
    ├── 脂肪组织：吸收葡萄糖、抑制脂肪分解
    └── 大脑：传递"饱了"的信号

空腹 → 血糖↓ → 胰岛素↓ → 胰高血糖素↑
    │
    ├── 肝脏：开始分解糖原释放葡萄糖
    ├── 脂肪组织：开始分解脂肪释放脂肪酸
    └── 肌肉：切换到脂肪酸氧化模式
```

**胰岛素不是坏东西——胰岛素抵抗才是问题。**

### 胰岛素抵抗：代谢崩塌的第一块多米诺

```
正常状态：
  少量胰岛素 → 细胞响应 → 血糖入细胞 → 血糖正常

胰岛素抵抗：
  大量胰岛素 → 细胞不响应 → 血糖进不去 → 血糖高 → 胰腺更努力 → 更多胰岛素
  → 细胞更不响应 → ……恶性循环
  → 最终：胰腺β细胞衰竭 → 2型糖尿病
```

**胰岛素抵抗的下游效应**：
- 肝脏：脂肪堆积 → NAFLD（非酒精性脂肪肝，中国成年患病率~30%）
- 血管：内皮损伤 → 动脉粥样硬化
- 大脑：脑胰岛素抵抗 → 认知功能下降（见第五章"阿尔茨海默=3型糖尿病"）
- 脂肪组织：慢性低度炎症 → 更多胰岛素抵抗 → 更恶性循环

**最被忽视的事实**：你可以在血糖完全正常的时候就已经有胰岛素抵抗。常规体检只查空腹血糖，不查空腹胰岛素。按 Peter Attia 的说法——"常规体检告诉你一切都正常的那天，可能就是你代谢崩塌第一块多米诺倒下的那天。"

## 1.4 线粒体：细胞的发电站

### 线粒体不是你以为的"细胞里的电池"

线粒体更像是**分布式能源工厂**，它不仅产生 ATP，还参与：
- 细胞凋亡（程序性死亡）的决策——什么时候细胞该自杀
- 钙离子信号传导——神经、肌肉、激素释放都依赖它
- 活性氧（ROS）的产生和清除——氧化应激的源头

### 线粒体功能障碍：从衰老到慢性病

```
健康线粒体：
  葡萄糖/脂肪酸 → 电子传递链 → H⁺梯度 → ATP合酶 → 大量ATP + 少量ROS

功能障碍线粒体：
  葡萄糖/脂肪酸 → 电子传递链堵塞 → 电子泄漏 → 大量ROS → 氧化损伤 → DNA突变
  → 更功能障碍 → 更多ROS → ……恶性循环
```

**线粒体功能障碍是衰老九大标志之一（Hallmarks of Aging）。** 它几乎参与所有慢性病的发病机制——从心脏病到阿尔茨海默。

### 怎么保护线粒体

1. **Zone 2训练（有氧基地区）**：在还能正常说话的最高强度运动30-60分钟→刺激线粒体生物合成
2. **间歇断食**：禁食14-16小时→自噬激活→清除功能不良的线粒体
3. **冷暴露**：棕色脂肪激活→线粒体产热（UCP1蛋白）→"好的"ROS信号→抗氧化防御↑
4. **避免超加工食品**：果糖和工业种子油→线粒体电子传递链过载→ROS↑

## 1.5 代谢灵活性：为什么有些人吃碳水不胖

### 代谢灵活性 = 身体在"烧糖"和"烧脂肪"之间自由切换的能力

```
代谢灵活性高的人：
  饭后 → 快速切换到葡萄糖氧化
  空腹 → 快速切换到脂肪酸氧化
  → 血糖稳、不易饿、精神好

代谢灵活性低的人（代谢不灵活≈胰岛素抵抗的早期信号）：
  饭后 → 葡萄糖处理缓慢
  空腹 → 脂肪酸氧化能力差
  → 血糖波动大、吃完饭犯困、饿了就受不了
```

**代谢不灵活是大多数慢性代谢病的前奏。** 它比"血糖高"出现得更早——体检报告一切正常，但你已经不能有效切换燃料了。

### 改善代谢灵活性的方法

- **减少进食频率**（不是减少食量）——每次吃饭都触发胰岛素→长期高频进食让胰岛素"疲劳"
- **力量训练**——肌肉是最好的血糖吸收器，更多肌肉=更大的糖原储存库
- **Zone 2训练**——直接增加线粒体脂肪酸氧化能力
- **充足的深度睡眠**——睡眠不足一晚就可以使胰岛素敏感度下降25-30%

## 1.6 断食的科学：自噬、NAD⁺、Sirtuins

### 断食不是"饿肚子减肥"

断食（不摄入热量>12小时）触发的生理过程完全不同于普通饥饿：

```
断食12-16小时：
  → 糖原耗尽 → 切换到脂肪酸/酮体模式 → 酮体是大脑的高效燃料

断食16-24小时：
  → 自噬（Autophagy）启动 → 细胞清理错误折叠蛋白、功能不良线粒体
  → 这是细胞层面的"大扫除+回收站"——相当于细胞自己在做垃圾分类

断食24-48小时：
  → NAD⁺水平上升 → Sirtuins（长寿蛋白）激活
  → mTOR通路↓（细胞生长信号降低） → 干细胞再生↑
```

### 自噬——2016年诺贝尔生理学奖的主题

大隅良典发现了自噬的分子机制。简单说：当细胞"饿"的时候，它不会坐以待毙——它会**自己吃自己**。但它吃的是最不重要的部分：错误折叠的蛋白质、功能不良的线粒体、累积的细胞垃圾。

**自噬的净效果**：清理垃圾+回收原料+更新细胞器。这不只是"清理"——是"翻新"。

### NAD⁺和Sirtuins——为什么 NAD⁺前体这么火

```
NAD⁺是细胞里最重要的辅酶之一。年轻时充足，随年龄下降。

Sirtuins是一类"长寿蛋白"，需要NAD⁺才能工作。
Sirtuins做的事：
  → SIRT1: 调节代谢、抑制炎症
  → SIRT3: 线粒体抗氧化防御
  → SIRT6: DNA修复、端粒维护

当NAD⁺下降 → Sirtuins"没燃料" → 衰老加速
```

**这解释了为什么NAD⁺前体(NMN/NR)成为补充剂行业的明星**——理论上，补NAD⁺→Sirtuins重新工作→衰老减速。但实际上：人体试验能确认的是NAD⁺水平上升和适度的代谢改善，**人类寿命延长的直接证据尚未获得。**[H008]

### 哪些断食方式有证据

| 方式 | 描述 | 证据水平 | 适合人群 |
|------|------|---------|---------|
| **时间限制进食(TRE)** | 每天只在8-10小时内进食 | 🟢 高 | 大多数人 |
| **5:2 轻断食** | 每周5天正常、2天500-600千卡 | 🟢 高 | 减重/代谢改善目标 |
| **隔日断食(ADF)** | 一天正常、一天禁食/极低热量 | 🟡 中 | 严格目标，但坚持率低 |
| **长时间断食** | 24-72小时 | 🟡 中 | 需医疗监督 |

**最重要的结论**：断食不是"越久越好"。TRE（每天8-10小时窗内进食）有最多的长期坚持数据，也是副作用最小的方案。

## 1.7 总结：代谢的三条铁律

1. **胰岛素是总开关。** 理解它怎么工作、怎么失灵——你就理解了90%的现代慢性病。胰岛素抵抗可以在血糖正常时就已经开始了。肌肉是最好的血糖吸收器——这解释了为什么力量训练和运动对代谢有"超出燃脂"的保护作用。

2. **线粒体 = 发动机。** 保护好线粒体=保护好每一个细胞的发电站。Zone 2训练+间歇断食+避免超加工食品，是目前最确定的三条保护路径。

3. **代谢灵活性 > 代谢速度。** "吃得少"不如"切换得快"。一个代谢灵活的瘦子（能自如切换糖/脂肪氧化）比一个代谢不灵活的瘦子（表面瘦但胰岛素抵抗）健康得多。体重秤不能告诉你代谢灵活性。
