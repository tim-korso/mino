### 插花的艺术 — 手机端快速录入 (02:00–03:57)

汤姆要求在手机上使用 ikebana，不停记录物品。三次迭代：

#### 第一次：局域网暴露（最简方案）
- `vite.config.ts` 加 `host: '0.0.0.0'`，手机同 WiFi 访问 `http://192.168.120.123:5173`
- 已有 viewport meta tag + Tailwind 移动端适配，无需额外改动
- DeepSeek API 走 Vite proxy，手机端也能调 AI
- **前提**：Mac 醒着 + 同 WiFi

#### 第二次：快速录入模式
- 在 `useAI.ts` 新增 `quickParseItem(text)`：自然语言一句话 → AI 提取结构化字段
- 新建 `QuickAdd.tsx` 组件：单行输入 + AI 解析 + 预览卡片 + 一键保存
- **降级策略**：没配 API Key 或 AI 失败时，本地关键词匹配（分类/价格/位置）兜底
- 嵌入 `ItemList.tsx` 顶部，搜索栏上方

#### 第三次：批量 + 语音
- 在 `useAI.ts` 新增 `batchParseItems(text)`：多行文本 → AI 返回 `QuickParseResult[]`
- 新建 `BatchQuickAdd.tsx`：
  - **多行 textarea**：每行一件物品，自动 grow
  - **语音输入**：Web Speech API（`SpeechRecognition`），`lang='zh-CN'`，`continuous=true`
  - 录音按钮 🎤 切换，结果实时附加到 textarea，句号处自动换行
  - 批量解析后展示 review 列表，每件可编辑/删除，一键「全部保存」
  - Fallback：无 API Key 时每行本地关键词解析
- 与 QuickAdd 并存：QuickAdd 在上（单件秒录），BatchQuickAdd 在下（默认折叠，点「批量录入/语音输入」展开）

**技术要点：**
- Web Speech API 浏览器原生支持，不上传音频，Safari/Chrome 移动端均可用
- `continuous: true` + `interimResults: true` 实现边说边出字
- Batch AI prompt 用 `max_tokens: 1024`，单次可解析 ~20 件
- 每件生成唯一 ID：`batch-${Date.now()}-${random4chars}`，避免碰撞
- 数据仍存 localStorage，手机和 Mac 各自独立

**改动的文件：**
- `ikebana/vite.config.ts`：加 `host: '0.0.0.0'`
- `ikebana/src/hooks/useAI.ts`：新增 `quickParseItem`、`batchParseItems`，导出 `QuickParseResult` 接口
- `ikebana/src/components/QuickAdd.tsx`：新建，单件快速录入

### macOS 备忘录批量导出 (13:00–13:30)

导出汤姆 3954 条备忘录，按时间线排列成 Excel 以便查看删减。

**方案选择：**
- AppleScript 逐条读 0.33s/条 → 全量 22 分钟，且连续大量调用被 macOS 风控断连
- 社区搜索找到 `apple-notes-parser`（Python），pipx 安装，直接读 SQLite + protobuf 解码

**TCC 权限发现：**
- Automation 权限（AppleScript 控制 Notes）≠ Full Disk Access（读受保护数据库）
- SQLite 数据库在 `~/Library/Group Containers/group.com.apple.notes/NoteStore.sqlite`
- `cp` 和 `sqlite3` 都被 TCC 拦截——需通过有 Full Disk Access 的工具程序化读取
- 汤姆给终端开了 Full Disk Access 后成功

**工作流：**
1. `pipx install apple-notes-parser`
2. `apple-notes-parser export all_notes.json` → 3954 条，~5 秒
3. Python + openpyxl 转 Excel（311KB），按创建时间降序
4. 注意：部分笔记含二进制字符（`[\x00-\x08\x0b\x0c\x0e-\x1f\x7f-\x9f]`），需清理才能写入 Excel

**产物：**
- `workspace/notes-migration/备忘录全量_按时间排列.xlsx`（311KB，7 列 + 状态下拉）
