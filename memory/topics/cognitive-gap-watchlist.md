## 修复进度定义

- **0-20%**：认知空白完整存在。市场基本忽视或错误定价。
- **20-40%**：早期信号出现。分析师开始覆盖或主流媒体开始正面报道。
- **40-60%**：空白正在闭合。财报连续验证，估值开始修复。
- **60-80%**：大部分已被定价。超额收益窗口收窄。
- **80-100%**：认知空白已修复。市场定价有效，Alpha消失。

## 范式智能（第四范式）6682.HK

### 认知空白
C端完全不知道这家公司存在。B端：连续七年中国机器学习平台市场份额第一(34%)，2025年营收71亿+35.6%，首次全年盈利，在手订单89亿超全年营收。股价从70跌到28（-60%），市场还在用旧标签。

### 修复路径：财报驱动

- [ ] **信号1**：Q2 2026继续保持Non-GAAP盈利（证明Q1不是"省出来"的一次性盈利）
- [ ] **信号2**：API业务收入连续两个季度超过2025全年（证明Token模式复利效应）
- [ ] **信号3**：ModelHub XC适配模型数突破10万（信创生态壁垒量化）
- [ ] **信号4**：毛利率企稳回升（证明标准化转型的成本结构改善）
- [ ] **信号5**：国际业务收入单独披露并显著增长

### 风险点
- 毛利增速(10.6%)远低于收入增速(35.6%)——标准化转型侵蚀短期毛利
- API业务基数小，高增速未必能持续
- 更名"范式智能"后市场认知混乱
- 港股中小市值流动性差

## DeepSeek 未上市

### 认知空白
C端认知："免费的AI聊天App"。现实：全球顶级开源基础模型实验室，MIT许可证1.6T参数模型权重随便下载商用，API定价是全球头部模型的1/10到1/100（不是补贴，是成本结构碾压），V4 Flash周调用量全球第一。

### 修复路径：叙事驱动（被动——梁文锋刻意不修）

- [ ] **信号1**：上市传闻或正式提交IPO申请（强制信息披露）
- [ ] **信号2**：V5发布，性能逼近或超越闭源SOTA
- [ ] **信号3**：C端付费产品推出并达到显著ARR
- [ ] **信号4**：梁文锋接受第一次公开采访（叙事重置）

### 风险点
- 梁文锋刻意保持沉默——C端认知空白是主动选择，不是失误
- 未上市，普通投资者无法直接投资
- 如果上市，极高估值可能已经预支了修复空间
- 人才流失问题（5名核心离职）可能影响V5进度

### 当前修复进度评估：5%
梁文锋不修。唯一能让C端认知重置的是：上市招股书/IPO路演强制披露，或V5的发布再次像R1那样"炸场"。
