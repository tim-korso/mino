## 技术教训

### 闲鱼网页版能力
- **goofish.com 搜索可用**（需登录），不像传言中完全阉割
- 搜索加载 JS 渲染搜索结果，等 3-5 秒即可
- 不登录返回"没找到宝贝"（非报错），登录后正常返回
- "个人闲置"过滤 + 手动识别可排除商家号和假货
- 网页版聊天可用，能直接发消息给卖家
- Playwright MCP + Chrome 浏览器 profile 持久化登录态

### iOS App 在 Mac 上的限制
- M 系列芯片可跑 iOS App，但无自动化接口
- 不如直接用网页版 + Playwright

### 自动化风险
- 正常浏览节奏 ≈ 真人，不触发风控
- 高频 API 抓取才会被检测
- 不要批量请求，控制翻页节奏
