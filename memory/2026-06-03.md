### 购物比价技术探索

#### Zyte API 评估
- Zyte = AI 驱动网页抓取平台（原 Scrapinghub），代理池+浏览器渲染+自动提取 JSON
- $5 免费额度 30 天，之后 $100/月起
- 对中文电商（淘宝/京东/拼多多）**无公开成功案例**，代理池主要覆盖欧美
- 两大集成路径：
  - **Zyte Claude Code Plugin**：`claude plugin marketplace add zyte-ai/claude-skills` → `claude plugin install zyte-web-data@zyte-ai`
  - **Zyte MCP Server**：`apscrapes/zyte-fetch-page-content-mcp-server`（Docker 部署，3 个工具）
- 结论：不适合中国电商比价场景

#### mcp-bijia 评估
- npm 包 `mcp-bijia@0.1.0`，MIT 协议，作者 eesire
- 设计方向正确：淘宝联盟/京东联盟/多多进宝 API → 价格排序+返利链接
- **实际状态**：骨架代码，所有搜索函数返回 `{available: false}` 的 stub 数据
- 三个工具定义好了（search_products / get_lowest_price / get_affiliate_link），核心逻辑全标 TODO
- 需要淘宝联盟/京东联盟 APPKEY 才能接入真实数据
- 结论：想法好但没实现，需要填核心 API 调用

#### 中国电商抓取现状
- 淘宝/拼多多反爬进入 AI 行为识别时代（浏览器指纹 50+ 维度、鼠标轨迹分析）
- 2025 年 4 月最高法判例：绕反爬爬取淘宝数据商用 → 判赔 500 万，定性不正当竞争
- 合法路径：官方联盟 API（淘宝联盟/京东联盟/多多进宝）——免费、稳定、合规
- 京东联盟个人可申，淘宝联盟需企业资质，多多进宝个人可申
