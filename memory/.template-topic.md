## Rules for Topic Files

1. **Quick Reference must stay under 10 lines** — it's the section everyone reads first
2. **Update the `last worked on` date** every time you touch the topic
3. **Add a row to Session History** every session that touches the topic
4. **Keep Known Limitations honest** — don't hide blockers
5. **Cross-link aggressively** — [[wikilinks]] are free discoverability
6. **When a topic file exceeds ~200 lines**, consider: is there a subtopic that should be its own file?
7. **Status transitions**: active → dormant (not currently working) → complete (finished, no active development) → archived (historical reference only)
