## Goal Loop 个人哲学框架

汤姆提出了 Goal Loop 的个人版本——一个关于目标循环与幸福的心理框架。

**三个点：**

1. **Goal Loop 本身** — 目标→追求→达成（或未达成）→新目标，永动不息。欲望驱动循环，循环不会停。
2. **两个痛苦源** — (a) 目标没达成时的不开心；(b) 认识到循环不会停——"我永远到不了终点"——这是一种存在主义级别的不安。
3. **翻转** — "我为什么要不开心？" 把这个问题问出来之后，反问本身改变了你在循环里的位置。从被循环推着走的人，变成看着循环在转的人。这个元认知距离就是第三个点。

**核心洞见**：循环不停不是诅咒，是设定。没有外部裁判。你在循环里选择高兴还是不高兴。

**与 Agent Goal Loop 的关系**：Agent 的 goal-loop 架构（见 `memory/topics/goal-loop.md`）是操作系统的设计——怎么让 Agent 持续产出。汤姆的 personal goal loop 是操作者的哲学——怎么在持续运转中保持心理健康。两层：**操作系统层（Agent）和操作者层（人）**。同源概念，不同应用。
